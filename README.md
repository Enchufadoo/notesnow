# Note Snow

### A sheet music note guessing game

Try it out at:

https://enchufadoo.github.io/notesnow

![](src/assets/images/logo.jpg?raw=true "")

Project setup
```
npm install
```

Compiles and hot-reloads for development
```
npm run serve
```

Compiles and minifies for production
```
npm run build
```

Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
